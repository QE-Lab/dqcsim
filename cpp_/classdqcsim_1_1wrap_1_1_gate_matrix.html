inlinestaticnoexcept

Computes the matrix for an arbitrary rotation.

The matrix is as follows:

\[ U(\theta, \phi, \lambda) = \begin{bmatrix} e^{i\frac{-\phi - \lambda}{2}} \cos{\frac{\theta}{2}} & e^{i\frac{-\phi + \lambda}{2}} \sin{\frac{\theta}{2}} \\ e^{i\frac{ \phi - \lambda}{2}} \sin{\frac{\theta}{2}} & e^{i\frac{ \phi + \lambda}{2}} \cos{\frac{\theta}{2}} \end{bmatrix} \]

This is equivalent to the following:

\[ U(\theta, \phi, \lambda) = R_z(\phi) \cdot R_y(\theta) \cdot R_z(\lambda) \]

The rotation order is taken from Qiskit's U3 gate, but the global phase is defined differently.

The rotation angles are specified in radians.

Parameters
thetaThe rotation angle in radians for the Y rotation.
phiThe rotation angle in radians for the pre-Y Z rotation.
lambdaThe rotation angle in radians for the post-Y Z rotation.
Returns
The matrix for a Z rotation gate with angle theta.
